Anti-Nuke

Im Detail

So arbeiten die Systeme

🚨

Anti-Raid

Automatisch · Bei Mass-Joins
  • Erkennt Mass-Joins (z.B. 10 Joins in 10 Sekunden)
  • Kickt / Bannt / Quarantänt alle Raider automatisch
  • Sperrt automatisch den ganzen Server (Lockdown)
  • Deaktiviert sich nach 5 Minuten automatisch
  • Whitegelistete User sind immun
☢️

Anti-Nuke

Überwachung · Audit-Log
  • Massen-Channel-Löschung (3+ Channels)
  • Massen-Rollen-Löschung (3+ Rollen)
  • Massen-Bans (5+ User gebannt)
  • Massen-Kicks (5+ User gekickt)
  • Spam-Channel/Rollen-Erstellung
  • Prüft Audit-Log für den Verursacher
  • Bannt Angreifer + entfernt kompromittierte Bots
📨

Anti-Spam

Echtzeit · Smart Detection
  • Schnelles Nachrichtenspammen (5 in 5s)
  • Mention-Spam (@everyone, viele Pings)
  • Emoji-Spam (zu viele Emojis)
  • Duplicate Messages (gleiche Nachricht)
  • Löscht Spam + Mute/Kick/Ban
🎣

Anti-Phishing

Immer aktiv · Link Scanner
  • Bekannte Phishing-Domains blockiert
  • Lookalike-Domains (d1scord, dlscord)
  • Homoglyph-Angriffe erkannt
  • "Free Nitro" + Link Pattern
  • Sofort gelöscht + 1h Timeout
  • DM-Warnung an den User
👶

Alt-Erkennung

Bei Join · Account-Check
  • Prüft Account-Alter bei jedem Join
  • Minimum: 7 Tage alt
  • Kick/Ban/Quarantäne bei zu jung
  • DM-Erklärung an den User
🔧

Auto-Moderation

Filter · Automatisch
  • Verbotene Wörter (konfigurierbar)
  • Discord-Einladungslinks blockiert
  • Alle Links (ohne Berechtigung)
  • GROSSSCHREIBUNG (70%+ Caps)
  • Löscht + warnt User
✔️

Verifikation

Button oder CAPTCHA
  • Button-Modus: Klick → Rolle
  • CAPTCHA-Modus: 6-stelliger Code
  • 3 Versuche erlaubt
  • Code läuft nach 2 Min ab
  • Auto-Kick nach 3 Fehlversuchen
⚠️

Auto-Warnsystem

Eskalation · Automatisch
  • 1-2 Warnungen → Nur gespeichert
  • 3+ Warnungen → 1h Auto-Mute
  • 5+ Warnungen → Auto-Ban

Wie richte ich den Bot ein?

Überwacht Audit-Log. Massen-Löschung von Channels/Rollen oder Massen-Bans → Angreifer wird sofort gestoppt.

Sind die Stats echt?

Ja, direkt von der Discord API. Aktualisierung alle 5 Minuten.
